# Ansible Multi-Task Automation Lab

This repository contains structured Ansible automation tasks performed in a DevOps lab environment.

## Tasks Covered

- Apache (httpd) installation & service management
- Web page deployment using blockinfile
- Archive creation using archive module
- ZIP extraction using unarchive module
- Permission enforcement (0655)
- Idempotent playbook design

## Execution

ansible-playbook -i inventory playbook.yml

## Concepts Practiced

- Multi-node configuration management
- Service lifecycle automation
- Infrastructure as Code (IaC)
